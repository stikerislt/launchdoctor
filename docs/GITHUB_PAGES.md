# GitHub Pages setup

Public marketing site: **https://stikerislt.github.io/launchdoctor/**

1. Repo → **Settings** → **Pages**
2. Source: branch **main**, folder **/docs**
3. Save

| Page | URL |
|------|-----|
| Home | `/` → `index.html` |
| FAQ | `/faq.html` |
| Privacy | `/privacy.html` |

Shared styles: `docs/assets/site.css`.

Update the install button URL in `docs/index.html` when your Shopify App Store listing is live (replace the App Store search link with your direct listing URL). Use `privacy.html` as the Privacy policy URL in the App Store listing.
